# SLÁN Rescue – Live Demo

Premium interactive demo of the SLÁN Public Safety Emergency App by Chosanta LTD.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Features

- **Citizen View** — Lock-screen RESCUE button, swipe gestures, triage wheel, two-stage confirmation, rich data packet, dispatch success
- **Command Dashboard** — Live incident feed, interactive map, data packet card, action buttons, citizen chat, status pipeline
- **Cinematic Demo** — One-click full sequence through citizen + command flow
- **PWA Ready** — manifest.json included for installable app
